## Overview
DataFlow Analytics is a browser-based data analysis and dashboard
planning tool that allows users to upload CSV or Excel files and
instantly explore their structure through KPIs, charts, and previews.

This project is designed to help analysts quickly understand datasets
and plan dashboards before building them in BI tools.

## Features
- CSV & Excel file upload
- Top / Bottom / Full dataset analysis
- Automatic KPI detection
- Logical visualizations:
  - Trend charts
  - Category comparisons
  - Scatter relationships
  - Distribution summaries
- Interactive tabs:
  - Overview
  - Charts
  - Data Preview
  - Insights
- Dashboard design guidance for:
  - Power BI
  - Tableau
  - Google Looker Studio

## Tech Stack
- HTML
- Tailwind CSS (CDN)
- JavaScript
- Chart.js
- SheetJS (xlsx)

## Notes
All processing is done client-side in the browser.
No backend or database is required.
